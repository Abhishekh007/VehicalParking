# Vehicle Parking Management System

## Project Description
A web-based Vehicle Parking Management System developed using PHP and MySQL.
It helps manage vehicle entry, exit, parking slots, and admin control.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Features
- Admin login
- Vehicle entry & exit
- Parking slot management
- Vehicle records
- Reports
